LIN总线管理API文档

1. 概述

该API提供了通过LIN总线控制方向盘按键和仪表盘视图的功能，支持GB和CLEA两种平台。

2. 主要类

LinGeneralManager

LIN总线管理的主类，提供方向盘按键控制和仪表盘视图切换功能。

2.2 仪表视图控制方法

check_view_result()

检查当前仪表视图状态。

返回: str - 当前视图名称

check_zone_result()

检查当前Zone2视图状态。

返回: str - 当前Zone2视图名称

cluster_view_by_lin(cluster_view, device=None)

切换仪表主视图。

参数:

  • cluster_view - 目标视图(从Constant.PATAC_SETTINGS_CLUSTER_VIEW_*中选择)
  • device - PCAN设备(可选)

返回: bool - 是否切换成功

cluster_zone2(zone2_view, device=None)

切换Zone2视图。

参数:

  • zone2_view - 目标视图(从Constant.PATAC_SETTINGS_ZONE2_VIEW_*中选择)
  • device - PCAN设备(可选)

返回: bool - 是否切换成功

cluster_zone3(zone3_view, device=None)

切换Zone3视图。

参数:

  • zone3_view - 目标视图(从Constant.PATAC_SETTINGS_ZONE3_VIEW_*中选择)
  • device - PCAN设备(可选)

返回: bool - 是否切换成功

3. 使用示例

3.1 方向盘按键控制示例

lin_manager = LinGeneralManager.getInstance()
lin_manager.switch3_Volume_Up()  # 音量加
lin_manager.switch_Cluster_View()  # 切换仪表视图

3.2 仪表视图切换示例

lin_manager = LinGeneralManager.getInstance()
# 切换到仪表视图
lin_manager.cluster_view_by_lin(Constant.PATAC_SETTINGS_CLUSTER_VIEW_GUAGE)
# 切换到Zone2的胎压视图
lin_manager.cluster_zone2(Constant.PATAC_SETTINGS_ZONE2_VIEW_TIREPRESSURE)

4. 注意事项

  • 使用前需要确保LIN总线连接正常
  • 不同平台(GB/CLEA)的LIN信号ID和名称可能不同
  • 视图切换可能需要多次尝试才能成功
  • 部分车型可能不支持所有视图
