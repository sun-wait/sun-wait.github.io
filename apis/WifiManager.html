WifiManager API 文档

单例模式： 该类采用单例设计模式，请使用 getInstance() 方法获取实例
底层ADB命令
  • ENABLE_WIFI_CMD: 启用WiFi
  • DISABLE_WIFI_CMD: 禁用WiFi
  • GET_WIFI_STATUS_CMD: 获取WiFi状态
  • CONNECT_WIFI_CMD: 连接WiFi网络
  • DISCONNECT_WIFI_CMD: 断开WiFi网络
  • GET_WIFI_LIST_CMD: 获取WiFi网络列表

EncryptionMode 枚举

定义WiFi加密模式：

  • wep: WEP加密
  • wpa: WPA加密
  • wpa2: WPA2加密 (默认)
  • wpa3: WPA3加密

WifiManager 类

该类提供了Android设备的WiFi管理功能，包括连接/断开WiFi、获取WiFi状态等。

__init__(self)
构造函数，初始化WifiManager实例。
getInstance()
获取WifiManager单例实例。
返回: WifiManager实例
isWifiEnable()
检查WiFi是否已启用。
返回: WiFi已启用返回True，否则返回False
switchWifi(on)
开启或关闭WiFi。
参数:
  • on - True开启WiFi，False关闭WiFi
返回: 操作成功返回True，失败返回False
getCurrentWifi()
获取当前连接的WiFi SSID。
返回: 当前连接的WiFi SSID，未连接返回None
getNetworkId(ssid)
获取指定SSID的网络ID。
参数:
  • ssid - 要查询的WiFi名称
返回: 网络ID，找不到返回None
disconnect(ssid)
断开指定WiFi连接。
参数:
  • ssid - 要断开的WiFi名称
返回: 操作成功返回True，失败返回False
connect(ssid, password, encryptMode=EncryptionMode.wpa2.value)
连接指定WiFi网络。
参数:
  • ssid - 要连接的WiFi名称
  • password - WiFi密码
  • encryptMode - 加密模式 (默认为WPA2)
返回: 连接成功返回True，失败返回False

使用示例

# 获取WifiManager实例
wifi_manager = WifiManager.getInstance()

# 开启WiFi
wifi_manager.switchWifi(True)

# 连接WiFi
wifi_manager.connect("MyWiFi", "password123")

# 获取当前WiFi
current_wifi = wifi_manager.getCurrentWifi()

# 断开WiFi
wifi_manager.disconnect("MyWiFi")
