Utils API 文档

class LogUtil

日志工具类，提供日志记录和打印功能，实现了单例模式。

类方法

@classmethod getInstance()

获取 LogUtil 的单例实例。

返回: LogUtil - LogUtil 的单例实例

实例方法

print(message, level="info")

在控制台打印消息并记录到日志文件。

参数:

  • message - 要打印的消息
  • level - 日志级别，可选"info"或"error"，默认为"info"
log(message)

记录错误日志，包含调用堆栈信息。

参数:

  • message - 错误消息
@staticmethod clearStack(stack)

清理堆栈对象。

参数:

  • stack - 要清理的堆栈对象
@staticmethod get_top_level_filename()

获取调用栈顶部的文件名。

返回: str - 文件名，失败返回None

class FileUtil

文件操作工具类，提供各种文件系统操作功能。

静态方法

isFileExist(path)

检查文件或目录是否存在。

参数:

  • path - 文件或目录路径

返回: bool - 是否存在

getLogDir()

获取最新的日志目录。

返回: str - 日志目录路径

getProjectDir(target_folder="automation_cases")

使用多线程在多个磁盘分区中查找目标文件夹。

参数:

  • target_folder - 要查找的目标文件夹名，默认为"automation_cases"

返回: str - 找到的完整路径，失败返回None

getParentDir(path)

获取指定路径的父目录。

参数:

  • path - 文件或目录路径

返回: str - 父目录路径

generateAbsPath(*params)

生成绝对路径。

参数:

  • *params - 路径组成部分

返回: str - 生成的绝对路径，失败返回None

getContent(path, encoding="utf-8")

读取文件内容。

参数:

  • path - 文件路径
  • encoding - 文件编码，默认为"utf-8"

返回: str - 文件内容，失败返回None

createFolder(path)

创建目录。

参数:

  • path - 目录路径
delete(filePath)

删除文件或空目录。

参数:

  • filePath - 文件或目录路径

返回: bool - 是否删除成功

writeFile(filePath, content, mode="w+", encoding="utf-8")

写入文件内容。

参数:

  • filePath - 文件路径
  • content - 要写入的内容
  • mode - 写入模式，默认为"w+"
  • encoding - 文件编码，默认为"utf-8"

返回: bool - 是否写入成功

deleteFolder(folder)

递归删除目录及其内容。

参数:

  • folder - 目录路径

返回: bool - 是否删除成功

listDir(path)

列出目录内容。

参数:

  • path - 目录路径

返回: list - 目录内容列表，失败返回None

isFile(file)

检查路径是否为文件。

参数:

  • file - 路径

返回: bool - 是否为文件

isDir(file)

检查路径是否为目录。

参数:

  • file - 路径

返回: bool - 是否为目录

copyFile(srcFile, dstFolder)

复制文件到目标目录。

参数:

  • srcFile - 源文件路径
  • dstFolder - 目标目录路径

返回: bool - 是否复制成功

moveFile(srcFile, dstFolder)

移动文件到目标目录。

参数:

  • srcFile - 源文件路径
  • dstFolder - 目标目录路径

返回: bool - 是否移动成功

getBaseName(file)

获取文件名（不含路径）。

参数:

  • file - 文件路径

返回: str - 文件名，失败返回None

gzipfile(srcFile, dstFile)

解压gzip文件。

参数:

  • srcFile - 源gzip文件路径
  • dstFile - 目标文件路径

返回: bool - 是否解压成功

get_filename_suffix(file_name)

获取文件后缀名。

参数:

  • file_name - 文件名或路径

返回: str - 文件后缀名，失败返回None

get_filename(file_path)

从路径中获取文件名。

参数:

  • file_path - 文件路径

返回: str - 文件名，失败返回None

zip(src_file, zip_file)

压缩文件或目录为zip格式。

参数:

  • src_file - 要压缩的源文件或目录
  • zip_file - 目标zip文件名

返回: str - 生成的zip文件路径，失败返回None

unzip(zip_file, dst_folder=None)

解压zip文件。

参数:

  • zip_file - zip文件路径
  • dst_folder - 目标解压目录，默认为zip文件所在目录

返回: str - 解压目录路径，失败返回None

getSize(file)

获取文件或目录大小（字节）。

参数:

  • file - 文件或目录路径

返回: int - 大小（字节）

全局变量

  • PROJECT_PATH - 缓存项目路径的字典
  • SEARCH_PROJECT_DIR_THREAD_EVENT - 线程事件对象
  • SEARCH_PROJECT_DIR_THREAD_LOCK - 线程锁对象

注意事项

  • LogUtil 是单例类，应通过 getInstance() 方法获取实例
  • FileUtil 的所有方法都是静态方法，无需实例化即可使用
  • 文件路径操作都使用 Path 对象处理，确保跨平台兼容性
  • 所有方法都有异常处理，失败时会记录日志并返回安全值
  • getProjectDir 方法使用多线程搜索磁盘，性能较高
