KeyEventer API 文档

单例模式：该类采用单例设计模式，请使用getInstance()方法获取实例。

KeyEventer 类

该类提供了发送按键事件的功能，支持发送按键码和预定义的按键事件。

__init__(self)
构造函数，初始化KeyEventer实例。

注意：不应直接实例化该类，请使用getInstance()方法获取实例。

getInstance()
获取KeyEventer单例实例。
返回: KeyEventer实例
sendKeyCode(self, device, keycode)
发送按键码到指定设备。

按键码可以是十六进制格式。

参数:
  • device - 目标设备对象
  • keycode - 要发送的按键码
sendKeyEvent(self, device, event)
发送预定义的按键事件到指定设备。

通常使用预定义的常量值作为事件参数。

参数:
  • device - 目标设备对象
  • event - 要发送的按键事件
使用示例： 
# 获取KeyEventer实例
key_eventer = KeyEventer.getInstance()

# 发送HOME键事件
key_eventer.sendKeyEvent(device, Constant.KeyEvent.home.value)

# 发送按键码
key_eventer.sendKeyCode(device, 0x03)
