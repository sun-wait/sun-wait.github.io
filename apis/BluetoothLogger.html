BluetoothLogger API 文档

该模块提供了收集蓝牙相关日志的功能，包括开启蓝牙日志记录和导出蓝牙日志文件。

命令说明

SET_BLUETOOTH_LOG_CMD = 'adb shell setprop persist.bluetooth.btsnooplogmode full'

设置蓝牙日志模式为完整记录模式。

DUMP_BLUETOOTH_LOG_CMD = "adb pull /data/misc/bluetooth {}/BluetoothLog"

导出蓝牙日志文件到指定目录。

DUMP_BLUEDROID_CMD = "adb pull /data/misc/bluedroid {}/BluetoothLog"

导出Bluedroid日志文件到指定目录。

CLEAR_BLUETOOTH_LOG_CMD = 'adb shell "echo -n > /data/misc/bluetooth/logs/btsnoop_hci.log"'

清空蓝牙HCI日志文件。

DELETE_BLUEDROID_CMD = "adb shell rm /data/misc/bluedroid/*"

删除Bluedroid目录下的所有文件。

类文档

BluetoothLogger

蓝牙日志收集器的主类，提供收集蓝牙相关日志的功能。

getInstance()
获取BluetoothLogger的单例实例。
返回:
BluetoothLogger的实例
startBluetoothLog()
开启蓝牙日志记录功能。
返回:
布尔值，表示是否成功开启
dumpBluetoothLog(caseFolder)
导出蓝牙日志文件到指定目录，并清理日志文件。
caseFolder 日志保存的文件夹路径
返回:
布尔值，表示是否成功导出

使用示例


# 获取BluetoothLogger实例
logger = BluetoothLogger.getInstance()

# 开启蓝牙日志记录
logger.startBluetoothLog()

# 进行蓝牙相关测试...

# 导出蓝牙日志到指定目录
logger.dumpBluetoothLog("/path/to/logs")

工作流程

  1. 调用startBluetoothLog()开启蓝牙日志记录
  2. 进行蓝牙相关测试操作
  3. 调用dumpBluetoothLog(caseFolder)导出日志文件
  4. 日志文件将被保存在caseFolder/BluetoothLog目录下
  5. 导出完成后会自动清理设备上的日志文件
