1. Power Cycle Stress Test (STRStressRunner)

1.1 基本用法


from power_cycle_stress_str_test import STRStressRunner

# 创建测试实例
test = STRStressRunner()

# 运行测试（默认24小时）
test.main()

1.2 方法详解

  • start_cluster_functions(): 启动集群功能 
    
# 启动集群功能（仪表、ADAS、环境光等）
test.start_cluster_functions()
  • stop_cluster_functions(): 停止集群功能 
    
# 停止所有集群功能
test.stop_cluster_functions()
  • start_monkey_test(): 启动Monkey测试 
    
# 启动Monkey测试
test.start_monkey_test()
  • stop_monkey_test(): 停止Monkey测试 
    
# 停止Monkey测试
test.stop_monkey_test()
  • teardown(): 清理测试环境 
    
# 清理测试环境
test.teardown()

2. System Pressure Test (SystemPressureRunner)

2.1 基本用法


from SystemPressureRunner import SystemPressureRunner

# 创建测试实例
stress = SystemPressureRunner()

# 设置并运行测试（默认24小时）
stress.setup(log_folder="your_log_path")
stress.main()

2.2 方法详解

  • startMonkeyTest(packageActivity=None): 启动Monkey测试 
    
# 启动Monkey测试
stress.startMonkeyTest()
  • stopMonkeyTest(): 停止Monkey测试 
    
# 停止Monkey测试
stress.stopMonkeyTest()
  • startClusterFunctions(): 启动集群功能 
    
# 启动集群功能（ADAS、仪表、警告等）
stress.startClusterFunctions()
  • stopClusterFunctions(): 停止集群功能 
    
# 停止所有集群功能
stress.stopClusterFunctions()
  • startExceptionChecker(module): 启动异常检查 
    
# 启动异常检查
stress.startExceptionChecker(module="performance")
  • stopExceptionChecker(): 停止异常检查 
    
# 停止异常检查
stress.stopExceptionChecker()
  • startRecordVideo(videoPath): 开始录制视频 
    
# 开始录制视频
stress.startRecordVideo(videoPath="your_video_path")
  • stopRecordVideo(): 停止录制视频 
    
# 停止录制视频
stress.stopRecordVideo()
  • teardown(): 清理测试环境 
    
# 清理测试环境
stress.teardown()

3. 主要区别

功能 STRStressRunner SystemPressureRunner
测试重点 电源循环稳定性 系统压力测试
集群功能 基础功能 更全面的功能
日志收集 基本日志 全面的日志收集（包括核心转储、视频等）
异常处理 基本处理 完善的异常处理机制

4. 选择指南

  • 如果需要测试电源循环稳定性，选择 STRStressRunner
  • 如果需要全面的系统压力测试，选择 SystemPressureRunner

5. 注意事项

  • 确保测试环境配置正确
  • 为日志收集预留足够的存储空间
  • 测试过程中避免手动干预
  • 测试完成后及时分析日志
