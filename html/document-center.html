模块文档

Infinium Test · 极测无界 项目包含多个功能模块，每个模块专注于特定领域的功能：

  • 通用模块 (Common) - 性能监控、数据采集和分析
  • 仪表模块 (Cluster) - 性能监控、数据采集和分析
  • 性能模块 (Performance) - 性能监控、数据采集和分析
  • 标定模块 (Calibration) - UI 自动化测试工具
  • OpenCV 模块 - 图像处理和计算机视觉功能
  • 日志模块 (Loggers) - 日志收集和管理
  • 音频模块 (Audio) - 音频处理和测试

API参考

通用 (Common)

  • CommandExcutor - 命令执行工具类
  • DeviceUtil - 设备工具类
  • KeyEventer - 发送按键事件工具类
  • ScreenRecorder - 屏幕录制工具类
  • ScreenShoter - 屏幕截图工具类
  • SerialClient - 调试串口工具管理类
  • TimeUtil - 时间工具类
  • Utils - 日志、文件工具类
  • VehicleTypeManager - 车辆类型管理类

音频 (audio)

  • AudioManager - 音频处理和测试

标定 (Calibration)

  • CalibrationManager - 读写系统标定

总线服务 (canbus)

  • CanBusClient - 启动、停止canbus 服务

仪表 (cluster)

  • ADASSettingManager - ADAS Setting类
  • ClusterGeneralManager - 仪表功能管理类
  • GpsSimulationGeneralMangager - GPS仿真类
  • LinGeneralManager - Lin发送接收类
  • MessageSender - 报文消息发送类
  • RvcManager - RVC 管理类
  • VehicleInitManager - 发送车辆初始化信号类

性能 (Performance)

  • BaseScenarioRunner - 基础场景运行基类
  • MemoryCpuManager - 内存和 CPU 数据采集类
  • IoMonitor - IO 监控类
  • StartupManager - 冷/暖/热启动管理类
  • PageSwitchManager - 页面切换管理类

UI自动化 (UIAutomator)

  • BasePage - 基础页面类
  • ElementLocator - 元素定位器
  • KeyOperation - 按键操作
  • WifiManager - WIFI 管理器

图像处理 (OpenCV)

  • ImageUtils - 图像处理工具类
  • Camera - 摄像头操作工具
